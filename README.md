# Kaho

[English](README.md) ・ 日本語

カホは、元気いっぱいで明るい chatbot な BotBone プラグインです。

## 使用法

まずはビルド。

```
dotnet build
```

BotBoneの /plugins フォルダに、[Kaho.dll](bin/Debug/netstandard2.1/publish/Kaho.dll) を導入
BotBone を起動するだけ
